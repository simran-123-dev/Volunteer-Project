# 🌟 NayePankh Foundation – AI Powered Volunteer Management System

An AI-powered Volunteer Management System built using the **MERN Stack** with **Google Gemini AI** integration. The platform helps NGOs manage volunteers, events, and user registrations while providing an intelligent AI assistant for volunteer recommendations and career guidance.

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/simran-123-dev/Volunteer-Project.git
```

```
cd Volunteer-Project
```

Install Dependencies

```bash
npm install
```

Install Client

```bash
cd client
npm install
```

Install Server

```bash
cd ../server
npm install
```

---

# ▶️ Run Project

### Backend

```bash
cd server
npm run dev
```

### Frontend

```bash
cd client
npm run dev
```

# 🌍 Environment Variables

## Server (.env)

```env
PORT=5000
MONGO_URI=YOUR_MONGODB_URI
GEMINI_API_KEY=YOUR_GEMINI_API_KEY
GEMINI_MODEL=gemini-2.5-flash
```

## Client (.env)

```env
VITE_API_URL=http://localhost:5000/api
```
